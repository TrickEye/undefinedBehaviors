## definition

The program attempts to modify a string literal.

程序尝试修改字符串字面量。

## description

字符串要求以'\0'结尾，字符串字面量无此要求（包含的范围更广）。

## code

```c
# include <stdio.h>

int main()
{
	char str[] = "We are happy.";
	str[12] = '!';	//undefined behavior
	printf("%s \n", str);
    return 0;
}
```
